# 📝 To-Do App
🚀 Esta es una aplicación sencilla pero funcional para gestionar tus tareas diarias.

---

### 🌟 Características
- Añadir nuevas tareas
- Marcar tareas como completadas
- Eliminar tareas

---

### 🛠️ Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- React + Vite
